# Component Libraries

Compound is built for both mobile and web. It supports three component libraries
for Android, iOS and Web.

## Compound Android

- Source code on [vector-im/element-x-android](https://github.com/vector-im/element-x-android/tree/develop/libraries/designsystem)
- Library built on top of Material 3 using Jetpack Compose
- Components preview using [Showkase](https://github.com/vector-im/element-x-android/blob/develop/docs/screenshot_testing.md)

## Compound iOS

- Source code on [vector-im/compound-ios](https://github.com/vector-im/compound-ios)
- Library built on top of SwiftUI
- Components preview available on TestFlight, reach out in [Compound | Lobby](https://matrix.to/#/#compound:matrix.org) to access

## Compound Web

- Source code on [vector-im/compound-web](https://github.com/vector-im/compound-web)
- Library built on top of Radix UI using React.js
- Components preview on [Storybook](https://compound-web.pages.dev/)
